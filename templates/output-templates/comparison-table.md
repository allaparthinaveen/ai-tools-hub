# Comparison Table

## Schema
```markdown
| Feature/Metric | Option A | Option B | Winner |
|----------------|----------|----------|--------|
| Cost           | $X       | $Y       | A      |
| Complexity     | High     | Low      | B      |
| Performance    | O(n)     | O(1)     | B      |
```

## Usage in Prompts
Add this to any prompt: "Respond using ONLY the table schema from `templates/output-templates/comparison-table.md`. No text before or after the table."
