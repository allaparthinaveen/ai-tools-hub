# Incident Postmortem

## Schema
```markdown
# Incident: [Date] [System Name] Outage

## 1. Summary
[Brief description of outage]

## 2. Timeline
- [HH:MM] - Detected issue
- [HH:MM] - Root cause identified
- [HH:MM] - Resolution pushed

## 3. Root Cause Analysis
[Technical explanation]

## 4. Action Items
- [ ] Task 1 (Assignee)
```

## Usage in Prompts
Add this to any prompt: "Respond using ONLY the schema from `templates/output-templates/incident-postmortem.md`"
