# Markdown Report

## Schema
```markdown
# [Title of the Report]

## Executive Summary
[High-level overview, 2-3 sentences max]

## Findings
- Sub-finding 1
- Sub-finding 2

## Conclusion / Next Steps
[Actionable next steps]
```

## Usage in Prompts
Add this to any prompt: "Respond using ONLY the schema from `templates/output-templates/markdown-report.md`"
