# AI Tools Hub Templates
This directory provides all the necessary configuration parameterization and orchestration layers to effectively run Prompts from the `/prompts/` directory.

## Structure
- `agent-config/`: YAML files linking system behavior & persona prompts.
- `prompt-templates/`: Markdown files that combine variables into coherent prompts.
- `workflow-templates/`: YAML orchestration pipelines outlining multi-step prompt chains.
- `output-templates/`: Strict schemas for programmatic AI outputs.
- `ui-templates/`: Configurations specifying UI rendering parameters for tools.
- `meta-templates/`: Formats for evaluation, logging, and versioning.
