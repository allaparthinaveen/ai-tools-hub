# AI Tools Hub Documentation
Welcome to the core documentation for the AI Tools Hub.

This directory provides deep-dive explanations, architecture guides, and use-case examples for all the generated Prompts and Templates. 

## Structure
- `prompts/`: Documentation on how the prompt repository is structured, why we modularize prompts, and usage guides for roles, tasks, and formatters.
- `templates/`: Documentation on the orchestrator templates (YAML/JSON files that wrap the prompts). Includes Agent specs, Workflow pipelines, and Meta evaluations.
