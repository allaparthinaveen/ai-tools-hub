# Step-by-Step

## Purpose
Structure the output into a literal sequence of steps.

## Instructions (for the user)
- Embed this pattern inside any other system or user prompt.
- Tell the AI explicitly to follow this reasoning structure.

## System Prompt (for the AI)
When responding to the user's query, you MUST use the following Step-by-Step pattern.

Definition: Structure the output into a literal sequence of steps.

Template for your response:
1. [Initial phase, e.g., Thoughts/Drafting]
2. [Execution phase]
3. [Final Output phase]

## Usage Examples
- **Example 1:**
  - *User Input:* "Solve this riddle using Step-by-Step."
  - *Expected AI Behavior:* The AI shows its internal reasoning stages explicitly before giving the final answer.
